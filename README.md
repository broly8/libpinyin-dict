# 安装

1. 下载需要的词库文件（比如“上市公司名称.txt”）**  
2. ibus-libpinyin 首选项 > 用户数据 > 导入**

您可以使用 fetch_pinyin.py 脚本为您自己准备的词汇清单自动生成拼音，然后导入。
